# open-resume

纯前端的在线简历编辑工具，支持导入导出 JSON 配置，导出 PDF 和图片

## 编译&运行
```shell
#编译镜像
docker build -t openresume .
#通过Compose启动
docker-compose up -d
#通过命令启动
docker run --it -name openresume -p 5173:5173 -d openresume
```
